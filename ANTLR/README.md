## ANTLR 簡介
ANTLR（ANother Tool for Language Recognition）使用規則來描述程式語言，  
利用 Lexer 將程式碼轉換成字元流、Parser 將字元流解析成抽象語法樹（AST）  
可以拿來做語言解析、語言查詢等應用
